请求返回值由两部分构成 status , message
status = 0成功 1出错 message存数据或报错信息, 无返回数据message 存 'success'

post 请求 ：
/api/signup 用户注册
{
    username,
    password,
    phone,
    category //非必填，默认为'common'
},

post 请求 ：
/api/login 用户登录
{
    username,
    password
}
返回token 解析token有用户id,name,category信息
